# Code Debugger Bot 

# Technologies Used:-

    OpenAI :- https://openai.com

    React Js :- https://react.dev


# Steps to install

    1- Run npm install in Project root directory.
    2- Add openAI Authorization Beared Token to App.js.
    3- Run npm start.
